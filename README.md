# 🧠 Online Quiz Web Application

A dynamic Online Quiz Web Application built using **Java, JSP, Servlets, and MySQL**.  
This project allows users to log in, attempt a quiz, and view their results instantly.

## 🚀 Technologies Used

- Java (JDK 8+)
- JSP
- Servlets
- MySQL
- Apache Tomcat Server
- Eclipse IDE

## 🎯 Features

User Login Authentication
Multiple Choice Questions
Instant Result Calculation
Clean UI using CSS
Session Handling using Servlets

## 🛠️ How to Run This Project (Eclipse + Tomcat)

 1️⃣ Import Project into Eclipse

1. Open Eclipse
2. Click File → Import
3. Select *Existing Projects into Workspace*
4. Browse and select the project folder
5. Click Finish

 2️⃣ Setup Apache Tomcat Server

1. Download Tomcat (Version 9 recommended)
2. In Eclipse → Go to **Servers**
3. Add New Server → Select Apache Tomcat
4. Add this project to the server
5. Start the server

 3️⃣ Setup MySQL Database

1. Open MySQL
2. Create a new database:
3. USE quizdb.sql code
4. Change username and password in LoginServlet.java file with yours.
